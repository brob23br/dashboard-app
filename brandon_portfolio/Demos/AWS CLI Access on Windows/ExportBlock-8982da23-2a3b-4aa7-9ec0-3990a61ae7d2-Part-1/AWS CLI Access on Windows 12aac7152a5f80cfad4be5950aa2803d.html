AWS CLI Access on Windows

Tags		AWSAWS Solutions Architect AssociateCAA-003Windowscommand line

To create an access key pair for AWS CLI access on Windows:

  1. Sign in to the AWS Management Console
  1. Navigate to the IAM (Identity and Access Management) dashboard
  1. Select the user for whom you want to create access keys
  1. In the user details page, go to the "Security credentials" tab
  1. Under "Access keys", click "Create access key"
  1. Choose "Command Line Interface (CLI)" as the use case
  1. Acknowledge the security recommendations and click "Next"
  1. (Optional) Add a description tag for the access key
  1. Click "Create access key"
  1. Download the CSV file containing the access key ID and secret access key

After creating the access key pair, configure the AWS CLI on your Windows machine:

$ aws configure
   AWS Access Key ID [None]: accesskey
   AWS Secret Access Key [None]: secretkey
   Default region name [None]: us-west-2
   Default output format [None]:

Enter your access key ID, secret access key, default region, and output format when prompted. This will set up your AWS CLI for use with your new access key pair.
