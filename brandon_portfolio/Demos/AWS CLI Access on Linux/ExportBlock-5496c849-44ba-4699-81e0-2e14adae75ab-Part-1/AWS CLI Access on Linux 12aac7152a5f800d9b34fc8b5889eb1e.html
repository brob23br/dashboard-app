AWS CLI Access on Linux

Tags		AWSAWS Solutions Architect AssociateCAA-003linuxmintterminal
  1. Navigate to aws.amazon.com/cli, then click the link for Linux Installer.
  1. Click the Linux dropdown.
  1. Scroll down and copy the code under “To Install the AWS CLI, run the following commands.”
  1. Paste into terminal, run the command. Verify the .zip file was downloaded by using ls.
  1. Unzip the file using the unzip command.
  1. Verify that the aws folder was made, then move to this folder, cd aws. Once in the aws directory, ls to find an install script.
  1. Run the install script with ./install. Verify version with /usr/local/bin/aws —version
  1. Now we need to run aws configure and apply our access key ID we made from https://www.notion.so/AWS-CLI-Access-on-Windows-119ac7152a5f80a881e7d2c4660521de?pvs=25 .
  1. Verify connection to your account with aws configure list, or access resources, in this case accessing buckets with /usr/bin/aws s3api list-buckets —query “Buckets[].Name”
